---

## 🔥 C'est quoi ?

GHOST1O1 Design System est un **design system hacker avant-gardiste** : composants UI, palette, typographie, motifs, animations — le tout pensé pour les outils de sécurité, les dashboards offensifs, les consoles Red Team.

**Pas de framework lourd** : CSS pur + variables, vanilla JS. Tu prends ce que tu veux, tu drop dans ton projet, ça marche.

---

## ✨ Features

- 🎨 **Palette NOCTURNE** : 12 couleurs signature
- 📝 **Typographie** : Mono + Sans + Display
- 🧩 **30+ composants** : cards, panels, tables, terminals, modals, alerts
- ♿ **Accessibilité** : WCAG 2.1 AA
- 🌗 **Dark-first** : conçu pour le confort visuel
- 📱 **Responsive** : cell → 4K
- ⚡ **Zero deps** : pas de React, pas de build, pas de bundler
- 📚 **100% documenté** : chaque composant a un exemple live

## 🚀 Démarrage 60 secondes

```html
<!-- Drop-in CSS -->
<link rel="stylesheet" href="https://cdn.ghost1o1.dev/design/v1.1/ghost1o1.css">

<!-- Utilise -->
<div class="ghost-card">
  <div class="ghost-header">
    <h3>NOCTURNE</h3>
  </div>
  <div class="ghost-body">
    <p class="ghost-text">Design system hacker.</p>
    <button class="ghost-btn ghost-btn-accent">ACTIVATE</button>
  </div>
</div>
```
